# QUICK INSTALLATION GUIDE
# DepEd Southern Leyte Division Library Log System

## STEP-BY-STEP INSTALLATION

### 1. PREPARE XAMPP
- Make sure XAMPP is installed.
- Start Apache and MySQL from XAMPP Control Panel.

### 2. CREATE DATABASE
1. Open browser: http://localhost/phpmyadmin
2. Click the "Import" tab.
3. Choose file: `database.sql`
4. Click "Go".
5. Database `library_logs` will be created with 2 tables:
- `users` (admin accounts)
- `logbook_entries` (visitor logs)

### 3. COPY FILES TO XAMPP
1. Go to: `C:\xampp\htdocs\` (or your XAMPP location)
2. Create folder: `library-system`
3. Copy all project files into this folder.

### 4. ACCESS THE SYSTEM
- Public Page: http://localhost/library-system
- Admin Login: http://localhost/library-system/admin/login.php

### 5. DEFAULT LOGIN
- Username: `admin`
- Password: `admin123`

## HOW IT WORKS

VISITOR SIDE:
- Visitor opens: http://localhost/library-system
- Fills in: Name, Client Type, Position, District, Purpose
- Clicks "SUBMIT LOG"
- Date and time are automatically recorded
- Single visit entry only (no time-out tracking)
- Done. No login needed.

ADMIN SIDE:
- Admin opens: http://localhost/library-system/admin/login.php
- Logs in with: admin / admin123
- Views all visitor logs in dashboard
- Can filter by date, name, or client type
- Can export logs to CSV
- Can delete entries if needed

## COMMON ISSUES

Problem: "Cannot connect to database"
Solution: Make sure MySQL is running in XAMPP

Problem: "Page not found"
Solution: Check if files are in correct folder (htdocs\library-system)

Problem: "Cannot login"
Solution: Make sure you imported database.sql first

Problem: "Table doesn't exist"
Solution: Re-import database.sql in phpMyAdmin
