# QUICK INSTALLATION GUIDE
# DepEd Southern Leyte Division Library Log System

## 1) START XAMPP
- Open XAMPP Control Panel
- Start `Apache` and `MySQL`

## 2) COPY PROJECT FOLDER
- Put this project inside: `C:\xampp\htdocs\`
- Example folder name: `library_logs`

## 3) IMPORT DATABASE
1. Open `http://localhost/phpmyadmin`
2. Go to `Import`
3. Choose `database.sql`
4. Click `Go`

Database `library_logs` and all required tables will be created.

## 4) OPEN THE SYSTEM
If folder name is `library_logs`:
- Public Page: `http://localhost/library_logs`
- Admin Login: `http://localhost/library_logs/admin/login.php`
- Master Data: `http://localhost/library_logs/admin/master_data.php`

## 5) DEFAULT ADMIN LOGIN
- Username: `admin`
- Password: `admin123`

Change password after first login.

## 6) WHAT ADMIN CAN MANAGE (DYNAMIC)
In `Master Data` page, admin can add/edit/activate/deactivate:
- Districts
- Schools
- Client Types
- Personnel

Master Data UX updates:
- Tabbed sections (Districts, Schools, Client Types, Personnel)
- Add/update actions save without full page refresh
- Floating toast messages + button save states (`Saving...` -> `Saved`)

## COMMON FIXES
- "Cannot connect to DB": Start MySQL and verify `config/database.php`
- "Page not found": Check folder name in URL
- "Cannot login": Import `database.sql` first
- "Master Data changes not using tabs/toasts": hard refresh browser (`Ctrl+F5`)
